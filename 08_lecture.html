  • Float wurde geschaffen damit man Text um Bilder herrum fließen lassen kann.
    Normalerweise verhält sich ein Bild (inline element) dass es die Zeile auf der es steht auf die gleiche größe wie das Bild selber zu vergrößern.
    Float left, steht links und lässt sich rechts umfließen
    Float right, steht rechts und lässt sich auf der linken Seite umfließen

  • Man könnte float elemente einfach in ein element hinein packen, dann wird das floaten von elementen ausserhalb des wrappers aufgehoben sein. Das problem dabei ist dass die box dann eine feste höhe haben muss denn die box kann die höhe nicht automatisch berechnen solange das floaten nicht aufgelöst wird.
    Die andere Lösung ist dem wrapper ein overflow: hidden; zu geben, das erlaubt dass der wrapper/box sich an dem errechnen der höhe versucht. Dies sorgt dafür dass das floaten die box garnicht erst verlässt.
