Notizen

  • Formular:
    definiert man wie folgt: <form action="#" method="post"> ... </form>
    wobei die attribute
    action: eine Aktion ist die beim Ausführen eines Submitbuttons ausgelöst wird. Dies kann ein Script sein.
    method: legt eine HTTP-methode fest nach der die daten geschickt werden.
  • Formularelemente
    ein formular kann mit verschiedenen Formularelementen gefüllt werden.
    Eine Liste dieser kann man unter Formulare und Formularelemente nachlesen.
  • Formulardaten an Server senden
    Hierzu legen wir eine php datei an wie zb submit.php
    in diese schreiben wir denn folgenden code: <pre> <?php print_r($_POST); ?></pre>
