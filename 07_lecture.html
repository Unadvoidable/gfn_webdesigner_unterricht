Notizen

  • tr ist eine table row, also eine spalte
    th ist ein table header
    td bedeutet table data.
  • Zellen lassen sich ebenfalls verbinden, dies geht immer vom td aus
    und zwar mit colspan (Horizontal verbinden) und rowspan (Vertikal verbinden)
    wichtig ist das das td auf die jenes angewendet wurde, die anzahl von plätzen einnimmt und deshalb muss man dementsprechend die drunter/nebenanliegenden Elemente löschen.
  • Tabellen nutzt man heutzutage nicht mehr zum designen, sondern nur wo man tabellen sonst benutzt. Bei daten etc. Aber auch bei Newslettern.
  • iFrame: Hiermit kann man andere Seiten in der Aktuellen Seite hineinladen/nachladen
    i in iframe steht für inline
  • per klick auf link einen neuen Inhalt in das bestehende iFrame hinein laden. Hierzu muss das ifram einen name="" zugewiesen bekommen
    Die links bekommen dann ein target="" zugewiesen. Somit wird der link die ihm zugewiesene Adresse in dem iFrame öffnen.
  • Manche Seiten haben das anzeigen ihrer Inhalte auf einem iFrame in einer anderen WEBseite untersagt
  • iFrames ist eine gute Alternative für Seiten mit vielen Menues wenn kein PHP interpreter auf einem WEbserver oder gar ein webserver verfügbar ist
  • iframes werden hauptsächlich dazu verwendet inhalte von andern webseiten anzuzeigen
  • youtube videos lassen sich zb. ebenfalls über ein iframe einbinden.
    Dazu geht man unter Teilen - Einbetten und dann bekommt man oft direkt den komplett den iFrame code
    bei google.maps funktioniert das genau so.
